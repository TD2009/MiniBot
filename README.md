OVERVIEW

This project is for a simple arduino bot, with the bare minimum to function. 
I threw together everything in an arduino kit to see what I could make, and this was the result.
It has the following at the moment: an IR Receiver, Joystick for manual testing, and 4 dc motors, controlled with an l298n.
It uses an Arduino R3.
The code can be used for other bots with a similar configuration, and can then be expanded on.
